# # How to run?

> 1. Clone repositiory
	
	git clone https://github.com/haibui2207/react-base-settings.git
	cd react-base-settings
> 2.  Install node package modules
	
	npm install
> 3.  Run project
	
	npm start
> 3.  Run styleguidist
	
	npm run styleguide
Access  `http://localhost:6060/` to view components.
